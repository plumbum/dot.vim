My dot.vim files
================


Usage
-----

  * git clone https://github.com/plumbum/dot.vim.git ~/.vim
  * ln -s ~/.vim/vimrc ~/.vimrc
  * Run vim
  * Install vundles :VundleInstall


YCM
---

  * ./install.sh --clang-completer --omnisharp-completer --system-libclang --system-boost
